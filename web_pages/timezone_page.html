Flask Time Zone Web app

Description

This web project utilized the time zone API to display the desired time zone to the user. This was done as a final project for Harvads CS50 course. This web app allows a user to login store there login in cookies and save desired time zones. All users are stored in an SQL data base.This was one of my First Flask applications.

  • Jinja
  • Python
  • JavaScript
  • HTML
