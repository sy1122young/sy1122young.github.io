Music Library

Description

This web project was for the University. I was in a team of four, we worked well together to create our Music Web app and got 13/15 mark. The music web app stores image paths in a JSON file, as well as a bio of the song and cover. The website also supports full search functionality

Technologies Used

  • PHP
  • JavaScript
  • HTML
  • CSS
  • JSON
  • Boot Strap library

Fetures

  • Asynchronous Requests for live updating
  • search capability
